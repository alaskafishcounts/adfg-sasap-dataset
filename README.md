# ADF&G SASAP Historical Dataset

## Overview

This dataset contains historical fish count data from Alaska Department of Fish and Game (ADF&G) monitoring stations across Alaska. The data spans multiple years and includes various salmon species counts from the State of Alaska Salmon and People (SASAP) project covering escapement data from 1921-2017.

## Geographic Areas

- **Active SASAP Stations:** 148 monitoring stations across Alaska
- **Species Coverage:** Chinook, Sockeye, Coho, Pink, and Chum salmon
- **Temporal Coverage:** Historical escapement data from 1921-2017

##  Data Format

All files follow the ADFG standard format with consistent column structure:

```json
{
  "COLUMNS": [
     "YEAR",
    "COUNTDATE",
    "FISHCOUNT",
    "SPECIESID",
    "COUNTLOCATIONID",
    "COUNTLOCATION",
    "SPECIES"
  ],
  "DATA": [
    [
      2002,
      "May, 26 2002 00:00:00",
      3,
      420,
      2000,
      "Karluk",
      "Sockeye"
    ]
  ],
  "metadata": {
    "location_id": 2000,
    "location_name": "Karluk",
    "species_id": 420,
    "species_name": "Sockeye Salmon",
    "year": 2002,
    "last_updated": "October 10, 2025T10:00:00Z",
    "data_source": "ADF&G SASAP"
  }
}
```

##  Regional Organization

All SASAP locations use AFCA IDs 20002999\. Regional grouping is not encoded in the ID range.
